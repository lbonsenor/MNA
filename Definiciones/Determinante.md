Es un valor escalar que se calcula a partir de los elementos de una matriz cuadrada. Sirve para saber si un sistema de ecuaciones tiene solución o para calcular áreas y volúmenes.
#### Propiedades
1. Una matriz $M^{n\times n}$ tiene columnas linealmente dependientes $\iff$ $\det(M)=0$ ^0486c8
2. $\det(M^{n\times n})\neq 0\implies$ La matriz es inversible $\implies$ sus columnas son linealmente independientes. ^5977be
